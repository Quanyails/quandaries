# Crossword Themes

- Actually shares a lot more with creative hobbies like art and writing. I.e., you should treat crossword ideas like a "sponge" in that they should be allowed to soak instead of wrung out. (I.e. "wait for the butterflies" principle.)
- Crossword construction has been around since the 1940s, so a lot of themes have been done already.
- A lot like research.
- The good news is that language constantly evolves, so new theme ideas appear over time.
- If your theme has been done before, you can salvage it by adding another "layer" to it.
